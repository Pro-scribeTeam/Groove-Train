## рҹҺ® Interactive Features
- **Hover Effect**: Yellow glow outline + tooltip showing region name

- **Erase Tool**: Remove color from any region
- **Clear All Button**: Reset entire painting
- **Save Artwork**: Download as SVG file
- **20+ Color Swatches**: Quick color selection
- **Paint/Erase Toggle**: Switch between tools
