## 🎮 Interactive Features
- **Hover Effect**: Yellow glow outline + tooltip showing region name
- **Click to Paint**: Fill with selected color (55% opacity for visibility)
- **Erase Tool**: Remove color from any region
- **Clear All Button**: Reset entire coloring page
- **Save Artwork**: Download as `my-train-boy-art.svg`
- **22 Color Swatches**: Quick color selection
- **Paint/Erase Toggle**: Switch between tools

- **SVG ViewBox**: 1024×1024 (1:1 aspect ratio)

- **Tooltip System**: Dynamic positioning follows mouse cursor
- **Fill Opacity**: 0.55 for painted regions (allows template visibility)
